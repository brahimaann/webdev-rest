Description of the Frameworks & API we used:

A little description of each of the frameworks and api that we used to create the St Paul Crime Map application

Leaflet

Leaflet is the leading open-source JavaScript library for mobile-friendly interactive maps. This library allows us to have a realtime map of the city where we can add markers and borders to show the outline of each neighborhood and entire city. This allowed us to physically visualize the data for the user to give them a better user experience.

Vue

Vue is a JavaScript framework for building user interfaces. It builds on top of standard HTML, CSS, and JavaScript and provides a declarative and component-based programming model that helps in developing user interfaces. We used Vue for automating and programming everything you see on our website. Through Vue, we were able to connect our API and work with it in live to to populate tables and the map. We also used Vue to modify our database by sending requests back to our API.

Our API

Our API is quite similar to others. We created our API for a project in our Web Development Class at University of St Thomas. Our API allows us to connect to our Database and request JSON objects back. Based on the URL path you input, you can either get incidents, add new crime incidents to our database or delete an incident based on the case number.

Interesting things we found using our application

The first thing we found super interesting was that Summit Hills had the least amount of crimes commited in all the neighborhoods!

Capitol River (AKA Downtown St.Paul) had the most amount of crimes commited in the latest 1000 reported crimes.

St Thomas University, located in Union Park had 74 out of the recent 1000 crimes reported there. That's 7.4%

University Avenue falls directly inbetween neighborhoods 7 and 8. Because of this, some crime statistics could be skewed since the street is in both neighborhood.

The Como neighborhood does not have as many reported crimes despite rumors of Como being a "rough" place.

Overall, the crime is quite evenly distrubted in St Paul. There are no major outliers in terms of crime.
